Historia

Este e um projeto realizado a partir de um desafio que foi nos passado chamado Assessment.

Como o desenvolvedor deste site comecou trabalhando diretamente com frameworks, lhe faltava base de HTML + CSS puro.

Estrutura

O projeto se inicia no arquivo index.html, onde fica sua pagina inicial.

Foi desenvolvido um esquema de diretorios e arquivos para que fique claro e coeso o entendimento da estrutura do projeto.

Desenvolvimento

Como mencionado na historia, o desenvolvedor responsavel por esse site nao possuia base alguma e lhe faltavam alguns conceitos das tecnologias utilizadas.

Com esse projeto essa base agora esta mais fortalecida e seu conhecimento ainda mais amplo, tendo em vista que esse conhecimento se torna necessario para seu objetivo de carreira.
